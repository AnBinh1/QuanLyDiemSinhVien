# 🎓 Hệ Thống Quản Lý Điểm Sinh Viên

Dự án **Quản Lý Điểm Sinh Viên** được xây dựng bằng **ASP.NET Core MVC** và **Entity Framework Core** nhằm hỗ trợ quản lý sinh viên, lớp học, điểm số và phân quyền người dùng (Admin/Sinh viên).  
Đây là một ứng dụng web phục vụ cho việc học tập, giảng dạy và nghiên cứu công nghệ ASP.NET.

---

## 🚀 1. Mục tiêu dự án

- Quản lý thông tin sinh viên, lớp, môn học và điểm số.  
- Quản trị người dùng, phân quyền sử dụng (Admin / Sinh viên).  
- Cung cấp giao diện thân thiện, dễ sử dụng và có thể mở rộng thêm các chức năng khác.

---

## 🧰 2. Công nghệ sử dụng

| Thành phần | Mô tả |
|-------------|-------|
| **Ngôn ngữ** | C# |
| **Framework** | ASP.NET Core MVC |
| **ORM** | Entity Framework Core |
| **CSDL** | SQL Server |
| **Frontend** | HTML, CSS, Bootstrap, Razor |
| **IDE** | Visual Studio 2022 |
| **Quản lý người dùng** | ASP.NET Identity |
